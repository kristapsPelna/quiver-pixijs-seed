# quiver-pixi-seed

A simple starter project using Quiver Framework and PIXI.js with a fullscreen renderer.

### Usage
- Clone or fork this repository
- Make sure you have [node.js](https://nodejs.org/) installed
- Make sure you have NPM installed version
- run `npm install` to install dependencies
- run `npm start` to start the dev server
- open browser to [`http://localhost:3000`](http://localhost:3000)
- to use other port, open `package.json` file, then change port in `--port 3000` script
